
# Shaun's Personal Site

A personal portfolio and showcase built with .NET 8 Blazor Server and MudBlazor. The goal is a clean, handable site — something I can point anyone at that gives them a real sense of who I am and what I work on.

The colour palette draws inspiration from Iceland's mossy volcanic terrain: muted greens, dark basalts, and the earthy warmth of the landscape.

## What's on the site

The site is organised into a few distinct sections, each covering a different area of my work and interests:

**Programming** — software projects spanning utilities and tools, learning experiments *(like a chess app)*, and game mods.

**Builds** — 3D printing projects split between functional/engineering pieces *(brackets, mounts, cable tidies)* and more creative or decorative work *(ornaments, display pieces)*. I really like physical making, even though I don't get to do it often, and this section is where that lives.

**Writing** — world-building and fiction writing. Each world has its own living wiki and story section. The source material stays private; only the compiled output is published here.

**Contact** — GitHub and LinkedIn. Nothing fancy, though I've stopped exposing my conteact email due to spam.

## Tech stack

- **Framework**: .NET 8 with Blazor Server
- **Component library**: MudBlazor
- **Language**: C#
- **Hosting**: Personal server with custom domain

## Running locally

```bash
git clone https://github.com/broomfields/blazor-personal-site.git
cd Blazor-Personal-Site
dotnet run
```

Requires the [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0). Once running, navigate to the URL shown in your terminal (typically `https://localhost:5001`).
